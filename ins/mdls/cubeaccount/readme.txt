== Description ==

CubeAccount Frontend Login lets your users login and register from the frontend of your site.

The WordPress dashboard and admin bar can be hidden completely from your users.
Users trying to reach wp-login.php directly would be redirected to your custom login page.

Installation is as easy as activating the plugin.

If permalinks are enabled, login and registration URLs would be rewritten to /login and /registration automatically.

Works well with other plugins that adds features to the login and registration form.

== Installation ==

Installation is as easy as uploading and activating the plugin.
